# Marketing Campaign Analysis

## Project Overview
This project analyzes marketing campaign data to evaluate which channels generate the highest ROI and conversion rates.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts / Dashboard

## Project Structure
The Excel file contains:
- Raw Data → Original dataset
- Cleaned Data → Data after cleaning and formatting
- Pivot Tables → Analysis of clicks, conversions, CTR
- Dashboard → Visual summary of KPIs

## Observation
- Social media campaigns generated the highest ROI
- Email campaigns produced the highest conversion rate
- The dashboard highlights trends over time for decision making

## Files Included
- `marketing_campaign_analysis.xlsx` → Full workbook with all sheets
- `dashboard_screenshot.png` → Quick preview of the dashboard
- `dataset.csv` → Raw data in CSV format
